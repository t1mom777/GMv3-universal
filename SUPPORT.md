# Support

## Getting help
- Open a GitHub issue for bugs, setup problems, and feature requests.
- Include OS, app version, logs, and exact steps to reproduce.

## Before opening an issue
- Check `README.md` for install and runtime requirements.
- Confirm API keys and provider settings in `.env`.
- Re-test with the latest release artifact.

## What to include
- Platform: Linux/Windows/macOS + version
- Build: release tag or artifact name
- Runtime mode: `voice-ws` / setup path
- Relevant logs and error messages
